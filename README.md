## Login Page UI

A modern and responsive login page built using **HTML** and **CSS**. This project focuses on clean design, structured layout, and an intuitive user interface without using JavaScript.

## Features

- Responsive design for different screen sizes
- Clean and minimal UI
- Styled input fields and buttons
- Simple and user-friendly layout
- Pure HTML & CSS implementation

## Project Structure
- **index.html**  
  Contains the structure of the login page including form fields, labels, and buttons.

- **style.css**  
  Handles all visual styling such as layout, colors, fonts, spacing, and responsiveness.

- **assets/** *(optional)*  
  Stores additional resources like images, icons, or custom fonts used in the project.
